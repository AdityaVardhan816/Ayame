# Ayame
A multifunctional utility and an RPG Discord bot with huge variety of commands to choose from, follows the rough sketch of the popular game series World of Warcraft.


Requires pre-installed packages mentioned as follows:
discord.py
aiohttp
praw
aiosqlite
google
beautifulsoup4
wikipedia-api
python-dotenv

You need to enable these in your server
SERVER MEMBERS INTENT (Bot uses member joins, roles, userinfo commands)
MESSAGE CONTENT INTENT (Bot uses text commands with prefixes)
PRESENCE INTENT (Check Spotify + custom activity in utils.py)

Requires these permissions:
Manage Messages (poof/purge, mute/unmute, lock/unlock)
Manage Roles (mute role creation)
Kick Members
Ban Members
Manage Channels (lock/unlock)
Embed Links
Read Message History
Send Messages
Use External Emojis
Add Reactions

Reddit API keys:
client_id
client_secret
user_agent
These MUST be moved to .env.

Automatically creates SQL tables if they dont exist automatically
